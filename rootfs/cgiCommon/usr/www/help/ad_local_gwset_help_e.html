Gateway Settings
MAC Address The MAC address of the router on the LAN, this value can not be changed
IP Address The IP address of the router on the LAN, the LAN default gateway for all devices must be set for the IP address
Subnet Mask The subnet mask of the LAN router, usually 255.255.255.0, all the computers in the LAN subnet mask must be set the same here
Note 1: if you change the IP address of the gateway, so you must be logged on using the new IP address to the WEB router routing management
2 if you have set the IP address of the gateway to new and original gateway IP address is not in the same network segment, the DHCP server IP will assign the router automatically change to segment the new gateway IP address, but the original IP network related functions, such as port mapping will fail.
