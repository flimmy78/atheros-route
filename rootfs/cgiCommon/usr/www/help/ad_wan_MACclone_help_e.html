MAC Address The Router's WAN MAC address, generally do not change this value. However, some ISP may require MAC address binding, then the ISP will provide a valid MAC address to a user, you can simply based on the value it provides input to the "MAC Address" field, then click "Apply" , you can change the router's WAN MAC address
Manually enter the MAC address Select this, you need to enter the MAC address in the address bar
Use the default MAC address Select this, you can restore the default settings of the router to the WAN MAC address factory
Use the current terminal MAC address Select this, you can manage the current terminal's MAC address to the "MAC address" column.(Note: terminal only in LAN can use this function)
