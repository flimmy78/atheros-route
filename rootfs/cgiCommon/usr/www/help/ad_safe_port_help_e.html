Port mapping

Port mapping
External Port The port number of internet service
Internal Port The port number of internal services
IP Address The IP address of device which running the internal service
Protocol The protocol of the service
Status The work status of the service
For example If your web server (Port 8080) IP address is 10.10.10.102, then you need to specify the port mapping table:
external port internal port IP address protocol state
8080 8080 10.10.10.102 TCP
Note: the remote routing management external default port 80 is access control inside the occupied
