Basic Settings
Network Name (SSID) Wireless LAN login name used for authentication , only authenticated users can access the wireless network
Hide Wifi If selected, equipment will not search to the wireless network name, if you need to connect the wireless network, you need to add the network manually in the equipment. If unchecked, can search to the wireless network equipment
WiFi switch If enabled, wireless network signal. If closed, wireless network signal does not exist
Channel The wireless signal as the data signal transmission media channel
Security Select the encryption, the device need to enter a password to access the wireless network
KEY Set the wireless network access password
Show Password selected, set the password in clear text display . Unchecked, then set a password in cipher text
Dual mode HBD routing supporting dual mode, WIFI can work at 2.4GHz and 5.0GHz band at the same time, can work alone in the two frequency bands. The advantages and disadvantages of dual mode: 1, 2.4GHz band crowds than 5.0GHz band, which was used by many of the space in the wireless network, such as wireless phone, home appliances etc.. 2, the higher the frequency, the transmission distance is short, the scope of work is narrow, the ability to penetrate solid worse, in the range of use indoor smaller.
