Basic Settings
Network Name (SSID) Wireless LAN login name used for authentication , only authenticated users can access the wireless network
Hide Wifi If selected, other devices will be unable to search for the wireless network name. If unchecked, other devices will be able to search to the wireless network
WiFi switch If enabled, access the wireless network hosts will be able to access the network. If turned off, access the wireless network hosts will not be able to access the network
Channel a wireless signal transmitted as a data signal transmission media channel selection range from 1 to 13
Security Select the encryption , you need to enter a password to access the other devices wireless network . If not encrypted , then other devices to automatically connect to the network
KEY Set the wireless network access password
Show Password selected, set the password in clear text display . Unchecked, then set a password in cipher text
