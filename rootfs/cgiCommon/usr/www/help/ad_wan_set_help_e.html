WAN Settings

WAN Settings Please select the appropriate connection type based on the way the Internet your Internet Service Provider (ISP) to offer. For example, if you can get automatically assigned by the DHCP server IP address from the ISP, select "Dynamic IP"; If you have a fixed IP address provided by your ISP , select "Static IP"; If you access the Internet for ADSL virtual dialing mode , select "PPPoE"
Button Function
Automatic detection If you do not know what the WAN port connection type you should choose, please click on the button to see the automatic detection of access provided by the ISP . Test results for reference only , please consult your ISP
Static IP
IP address The ISP's DHCP server dynamically obtain an IP address
Subnet Mask The DHCP server dynamically from the ISP to get the subnet mask
Default Gateway The ISP's DHCP server dynamically get the gateway
Primary DNS server The DHCP server dynamically from the ISP to get DNS server , or you can set up your own DNS server manually here
Secondary DNS server Secondary DNS server from the ISP's DHCP server dynamically get, or you can set up your own DNS server backup manually here
Dynamic IP
Dynamic IP Internet access through Modem, ISDN, ADSL, cable, residential broadband, equipment every time the Internet assigned IP address is not the same, this is a dynamic IP address acquisition
PPPoE
User Name Fill your ISP assigned to ADSL Internet user name , your ISP may not know to ask
Internet Password Fill ISP ADSL Internet access you specify a password , it is unclear to ask the ISP
Operating Mode Keep Alive： If you choose to connect automatically at boot automatically after the connection to operate. During use , if due to external reasons , the network is disconnected , then every so often ( 10 seconds ) attempts to connect until the connection so far
on Demand： If you select Connect on Demand , then there is a network access request from the LAN to automatically connect operation
Manual： ADSL dial-up connection after boot performed manually by the user
Redial period： If the automatic disconnection waiting time T is not equal to 0, then no detected within minutes of continuous T network access when traffic is automatically disconnected from the network , protecting your online resources. This setting is only "on-demand connection" and " manual connection " effect
Timing：If you select a timed connection in the " connection time" specified start time to connect automatically operate automatically disconnected at the designated end time
