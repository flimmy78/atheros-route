Wireless advanced settings
SSID time The beacon interval, the default value is 100. Interval adjustable high, is helpful to develop wireless network performance; interval down, can speed up the wireless client connection speed
RTS/CTS limited value Set the maximum length of the data packet, the packet length exceeds the RTS/CTS limit, the router sends RTS to the destination site to negotiate, the default value is 2346
Header mode Send data packets between the interval, the default for short header mode. When the multipath effect is not very serious, recommended to choose the short header mode, data transmission rate of the router has higher. But for the multipath effect is obvious environmental, recommend the use of long header mode
Note: wireless advanced settings on this router has been configured to the best. If there is no support staff guidance, not to change the. If the change is inappropriate, there may be an accident, resulting in the router is unable to work
